# gorilla-cars

To view and make edits to this project, make sure you have npm and node js installed on your device.
  To install these dependencies, use these links:
    https://docs.npmjs.com/downloading-and-installing-node-js-and-npm 

To start, create a new project in a folder using this command in the terminal
  % npx create-next-app@latest 
Then, select these following prompts -
  √ Would you like to use TypeScript? ... No / **Yes**
  √ Would you like to use ESLint? ... No / **Yes**
  √ Would you like to use Tailwind CSS? ... No / **Yes**
  √ Would you like your code inside a `src/` directory? ... No / **Yes**
  √ Would you like to use App Router? (recommended) ... No / **Yes**
  √ Would you like to use Turbopack for next dev? ... **No** / Yes
  √ Would you like to customize the import alias (@/* by default)? ... **No** / Yes

To finish, replace the entire src folder inside with the one here.

Then, run
  npm run dev
and in your browser, enter the link it gives you -
  example: http://localhost:3000/
